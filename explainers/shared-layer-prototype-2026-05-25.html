02 The wall that matters — proven, not promised

Codex's worry was a client seeing another client's data. The fix isn't a rule we hope everyone follows — it's structural: an agent can only read messages the router physically addressed to it.

DAG repo → its own client
Subscribed to client "dagdc" only.

DAG repo → TDB's data
Tried adversarially in the test.
✕ physically cannot reach it
